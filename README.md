# Kitchen Companion

A companion app that keeps track of what you have in your food stock and generate shopping lists for your receipts.

# Current status

I'm currently starting the project so there is not a lot of things in here.
My goal is to implement an app that is going to be available for as much platforms as possible.
For now on, I will mainly focus on the Android and Linux platforms and, thanks to the frameworks that I use and once the app will be released for the two platforms, I project to add support for iOS, Windows and MacOS.

The final project is supposed to use Flutter for the mobile app and PyQt for the desktop app.
Since I'm still learning a lot of things, the PyQt part may change throughout the development.

Finally, I'm only working on the desktop app right now since Flutter is still a little bit complicated for me.
